WHAT IT DOES
A real agent, not a chatbot.

★ Self-Growing Skill Forge

Ask it to "learn to…" something. It writes the code, safety-checks it, asks you, then keeps it forever.

Real laptop control

Opens and operates apps, volume, media, the web, and reads & clicks any app via the macOS accessibility tree.

Voice-first HUD

A reactive command center — speak naturally, watch a live system log, surface news or the AI feed on demand.

World & tech intel

Live global news, markets, Indian economy, and a daily AI digest from Reddit, HN, arXiv and YouTube.

Resilient free brain

Rotates Groq → Cerebras → OpenRouter → Gemini so it keeps going when any provider throttles.

Safe by construction

Scoped folders, preview + one-tap confirm, Trash-only deletes, undo, and a scanner gating all forged code.

GET STARTED
Run it in two minutes.
git clone https://github.com/rithick007/jarvis
cd jarvis
cp .env.example .env      # add one free key (Groq is enough)
./run.sh                  # opens http://127.0.0.1:8000
  1. Grab a free key — Groq, Cerebras, or OpenRouter. More keys = more uptime.
  2. ./run.sh installs everything, builds the HUD, and opens it.
  3. Hit Start Talking and say "what's new in AI" or "open Spotify".

This is a local assistant: it controls your own Mac and runs on your machine, so it isn't a public web app — this page is the showcase, the agent lives on your computer.
