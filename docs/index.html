WHAT MAKES IT DIFFERENT

★ Self-Growing Skill Forge

Ask it to "learn to…" something it can't do. It writes the code, runs a deterministic safety scan + sandbox test, asks you to approve, then keeps the skill forever. Every install becomes unique.

Real laptop control

Opens and operates apps, controls volume/brightness/media, searches the web, and reads & clicks any app through the macOS Accessibility tree — behind a tiered safety layer.

Voice-first HUD

A reactive command center: speak naturally, watch a live system log, and surface world news or the daily AI feed on demand.

World & tech intelligence

Live global news, war/economy, Indian markets, plus a daily AI digest from Reddit, Hacker News, arXiv and YouTube — summarized by the model.

Resilient free brain

Rotates across Groq → Cerebras → OpenRouter → Gemini (all free tiers) with a browser fallback, so it keeps going when any provider throttles.

Safe by construction

Scoped folders, preview + one-tap confirm, Trash-only deletes, undo, and a static scanner gating every line of forged code.

RUN IT (LOCAL)

git clone https://github.com/rithick007/jarvis
cd jarvis
cp .env.example .env      # add one free key (Groq is enough)
./run.sh                  # opens http://127.0.0.1:8000
  1. Grab a free key — Groq, Cerebras, or OpenRouter. More keys = more uptime.
  2. ./run.sh installs everything, builds the HUD, and opens it.
  3. Click Start Talking and say "what's new in AI" or "open Spotify".

This is a local assistant: it controls your own Mac and runs on your machine, so it isn't a public web app — this page is the showcase, the agent lives on your computer.
